Introduction to Functions

A function is a block of code that performs a specific task. Functions help break our program into smaller and modular chunks, making it more organized, manageable, and reusable.

Function Declaration and Definition

A function declaration tells the compiler about a function's name, return type, and parameters. A function definition provides the actual body of the function.

// Function declaration
returnType functionName(parameterList);

// Function definition
returnType functionName(parameterList) {
    // function body
    // statements
    return value; // optional
}

Example:

#include <iostream>
using namespace std;

// Function declaration
int add(int a, int b);

int main() {
    int sum = add(5, 3);
    cout << "Sum: " << sum;
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

Function Parameters

Parameters are variables used in a function to receive values from the caller. There are three ways to pass parameters to a function:

Pass by Value

In this method, the actual value is passed to the function. Changes made to the parameter inside the function do not affect the original value.

#include <iostream>
using namespace std;

void modifyValue(int x) {
    x = x + 5; // Changes only the local copy
    cout << "Inside function: " << x << endl;
}

int main() {
    int num = 10;
    cout << "Before function call: " << num << endl;
    modifyValue(num);
    cout << "After function call: " << num << endl;
    return 0;
}

Output:

Before function call: 10
Inside function: 15
After function call: 10

Pass by Reference

In this method, the memory address of the variable is passed to the function. Changes made to the parameter inside the function affect the original value.

#include <iostream>
using namespace std;

void modifyValue(int &x) {
    x = x + 5; // Changes the original value
    cout << "Inside function: " << x << endl;
}

int main() {
    int num = 10;
    cout << "Before function call: " << num << endl;
    modifyValue(num);
    cout << "After function call: " << num << endl;
    return 0;
}

Output:

Before function call: 10
Inside function: 15
After function call: 15

Pass by Pointer

In this method, the memory address of the variable is passed to the function using pointers. Changes made to the parameter inside the function affect the original value.

#include <iostream>
using namespace std;

void modifyValue(int *x) {
    *x = *x + 5; // Changes the original value
    cout << "Inside function: " << *x << endl;
}

int main() {
    int num = 10;
    cout << "Before function call: " << num << endl;
    modifyValue(&num);
    cout << "After function call: " << num << endl;
    return 0;
}

Output:

Before function call: 10
Inside function: 15
After function call: 15

Return Values

Functions can return values using the return statement. The return type is specified in the function declaration.

#include <iostream>
using namespace std;

int square(int x) {
    return x * x;
}

int main() {
    int num = 5;
    int result = square(num);
    cout << "Square of " << num << " is " << result;
    return 0;
}

Default Parameters

C++ allows you to specify default values for function parameters. If a value is not provided for a parameter with a default value, the default value is used.

#include <iostream>
using namespace std;

void printMessage(string message = "Hello, World!") {
    cout << message << endl;
}

int main() {
    printMessage(); // Uses default parameter
    printMessage("Custom message"); // Uses provided parameter
    return 0;
}

Output:

Hello, World!
Custom message

Function Overloading

Function overloading allows you to have multiple functions with the same name but different parameters.

#include <iostream>
using namespace std;

// Function with two integer parameters
int add(int a, int b) {
    return a + b;
}

// Function with three integer parameters
int add(int a, int b, int c) {
    return a + b + c;
}

// Function with two double parameters
double add(double a, double b) {
    return a + b;
}

int main() {
    cout << add(5, 3) << endl;        // Calls add(int, int)
    cout << add(5, 3, 2) << endl;     // Calls add(int, int, int)
    cout << add(5.2, 3.7) << endl;    // Calls add(double, double)
    return 0;
}

Output:

8
10
8.9

Inline Functions

Inline functions are a C++ enhancement feature that improves program execution speed. When a function is declared as inline, the compiler places a copy of the function's code at each point where the function is called.

#include <iostream>
using namespace std;

inline int square(int x) {
    return x * x;
}

int main() {
    cout << square(5);
    return 0;
}

Recursive Functions

A recursive function is a function that calls itself. Recursive functions can be used to solve problems that can be broken down into smaller, similar subproblems.

#include <iostream>
using namespace std;

int factorial(int n) {
    if (n <= 1) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

int main() {
    int num = 5;
    cout << "Factorial of " << num << " is " << factorial(num);
    return 0;
}

Output:

Factorial of 5 is 120

Function Templates

Function templates allow you to create a function that can work with different data types. The compiler generates the appropriate function definitions for the data types used when the function is called.

#include <iostream>
using namespace std;

template 
T maximum(T a, T b) {
    return (a > b) ? a : b;
}

int main() {
    cout << maximum(5, 10) << endl;       // int version
    cout << maximum(5.5, 10.5) << endl;   // double version
    cout << maximum('a', 'z') << endl;    // char version
    return 0;
}

Output:

10
10.5
z

Lambda Functions

Lambda functions (introduced in C++11) are anonymous functions that can be defined inline. They are particularly useful for short functions that are used only once.

#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main() {
    vector numbers = {1, 2, 3, 4, 5};
    
    // Lambda function to print each number
    for_each(numbers.begin(), numbers.end(), [](int n) {
        cout << n << " ";
    });
    
    return 0;
}

Output:

1 2 3 4 5
