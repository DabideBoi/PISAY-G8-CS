Introduction to Loops

Loops are used to execute a block of code multiple times. C++ provides several types of loops to handle looping requirements.

For Loop

The for loop is used when you know in advance how many times you want to execute a statement or a block of statements:

for (initialization; condition; increment/decrement) {
    // code to be executed
}

Example:

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 5; i++) {
        cout << i << "\n";
    }
    
    return 0;
}

Output:

0
1
2
3
4

For Loop Components

  1. Initialization: Executed once before the loop starts. Often used to initialize a counter variable.
  2. Condition: Evaluated before each loop iteration. If true, the loop continues; if false, the loop ends.
  3. Increment/Decrement: Executed after each loop iteration. Often used to update the counter variable.

While Loop

The while loop loops through a block of code as long as a specified condition is true:

while (condition) {
    // code to be executed
}

Example:

#include <iostream>
using namespace std;

int main() {
    int i = 0;
    
    while (i < 5) {
        cout << i << "\n";
        i++;
    }
    
    return 0;
}

Output:

0
1
2
3
4

Do-While Loop

The do-while loop is a variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true:

do {
    // code to be executed
} while (condition);

Example:

#include <iostream>
using namespace std;

int main() {
    int i = 0;
    
    do {
        cout << i << "\n";
        i++;
    } while (i < 5);
    
    return 0;
}

Output:

0
1
2
3
4

Key Difference Between While and Do-While

The key difference between while and do-while is that do-while evaluates its condition at the end of the loop, which means the code block is executed at least once.

Nested Loops

You can also have loops inside loops, which are called nested loops:

#include <iostream>
using namespace std;

int main() {
    // Outer loop
    for (int i = 1; i <= 3; i++) {
        cout << "Outer: " << i << "\n";
        
        // Inner loop
        for (int j = 1; j <= 3; j++) {
            cout << " Inner: " << j << "\n";
        }
    }
    
    return 0;
}

Loop Control Statements

C++ provides statements that can be used to control the flow of loops:

Break Statement

The break statement can be used to jump out of a loop:

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i == 4) {
            break; // Exit the loop when i is 4
        }
        cout << i << "\n";
    }
    
    return 0;
}

Output:

0
1
2
3

Continue Statement

The continue statement breaks one iteration in the loop and continues with the next iteration:

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i == 4) {
            continue; // Skip the rest of the code in this iteration
        }
        cout << i << "\n";
    }
    
    return 0;
}

Output:

0
1
2
3
5
6
7
8
9

Infinite Loops

An infinite loop occurs when a condition in a loop always evaluates to true, causing the loop to run indefinitely:

// Infinite for loop
for (;;) {
    // code to be executed
}

// Infinite while loop
while (true) {
    // code to be executed
}

// Infinite do-while loop
do {
    // code to be executed
} while (true);

Infinite loops are generally avoided, but they can be useful in certain scenarios, such as in game development or when creating server applications that need to run continuously. In such cases, there should be a way to exit the loop based on certain conditions.
