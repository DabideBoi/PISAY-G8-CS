Introduction to JavaFX

JavaFX is a set of graphics and media packages that enables developers to design, create, test, debug, and deploy rich client applications that operate consistently across diverse platforms. JavaFX is intended to replace Swing as the standard GUI library for Java SE.

Key features of JavaFX include:

  • Modern UI controls and CSS styling
  • Hardware-accelerated graphics pipeline
  • FXML for UI design separation
  • Scene Builder for visual layout design
  • Rich set of UI controls and charts
  • Support for 2D and 3D graphics
  • Media playback support
  • Web content integration

JavaFX Architecture

JavaFX applications follow a specific architecture:

The main components of JavaFX architecture are:

  • Stage: The top-level JavaFX container (window)
  • Scene: The container for all content
  • Scene Graph: A hierarchical tree of nodes representing the visual elements
  • Nodes: The visual elements that make up the scene graph

Getting Started with JavaFX

Setting Up JavaFX

Since Java 11, JavaFX is no longer bundled with the JDK. You need to add it as a dependency to your project:

For Maven:

<dependencies>
    <dependency>
        <groupId>org.openjfx</groupId>
        <artifactId>javafx-controls</artifactId>
        <version>17.0.2</version>
    </dependency>
    <dependency>
        <groupId>org.openjfx</groupId>
        <artifactId>javafx-fxml</artifactId>
        <version>17.0.2</version>
    </dependency>
</dependencies>

For Gradle:

dependencies {
    implementation 'org.openjfx:javafx-controls:17.0.2'
    implementation 'org.openjfx:javafx-fxml:17.0.2'
}

Basic JavaFX Application

Here's a simple JavaFX application that displays a window with a button:

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.layout.StackPane;
import javafx.stage.Stage;

public class HelloWorldApp extends Application {
    
    @Override
    public void start(Stage primaryStage) {
        // Create a button
        Button btn = new Button("Say Hello");
        
        // Set button action
        btn.setOnAction(event -> System.out.println("Hello, JavaFX!"));
        
        // Create a layout and add the button to it
        StackPane root = new StackPane();
        root.getChildren().add(btn);
        
        // Create a scene with the layout
        Scene scene = new Scene(root, 300, 250);
        
        // Set the scene on the stage
        primaryStage.setTitle("Hello JavaFX");
        primaryStage.setScene(scene);
        
        // Show the stage
        primaryStage.show();
    }
    
    public static void main(String[] args) {
        // Launch the application
        launch(args);
    }
}

JavaFX UI Components

Layout Panes

JavaFX provides various layout panes to organize UI components:

  • BorderPane: Divides the space into top, right, bottom, left, and center regions
  • HBox: Arranges components in a single horizontal row
  • VBox: Arranges components in a single vertical column
  • GridPane: Arranges components in a flexible grid of rows and columns
  • FlowPane: Arranges components in a flow that wraps at the boundary
  • StackPane: Stacks components on top of each other
  • AnchorPane: Anchors components to the edges of the container
import javafx.application.Application;
import javafx.geometry.Insets;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.control.Label;
import javafx.scene.layout.BorderPane;
import javafx.scene.layout.HBox;
import javafx.scene.layout.VBox;
import javafx.stage.Stage;

public class LayoutExample extends Application {
    
    @Override
    public void start(Stage primaryStage) {
        // Create a BorderPane
        BorderPane borderPane = new BorderPane();
        
        // Create components for each region
        Label topLabel = new Label("Top Region");
        HBox topRegion = new HBox(topLabel);
        topRegion.setStyle("-fx-background-color: lightblue; -fx-padding: 10px;");
        
        Label leftLabel = new Label("Left Region");
        VBox leftRegion = new VBox(leftLabel);
        leftRegion.setStyle("-fx-background-color: lightgreen; -fx-padding: 10px;");
        
        Label centerLabel = new Label("Center Region");
        VBox centerRegion = new VBox(centerLabel);
        centerRegion.setStyle("-fx-background-color: white; -fx-padding: 10px;");
        
        Label rightLabel = new Label("Right Region");
        VBox rightRegion = new VBox(rightLabel);
        rightRegion.setStyle("-fx-background-color: lightgreen; -fx-padding: 10px;");
        
        Label bottomLabel = new Label("Bottom Region");
        HBox bottomRegion = new HBox(bottomLabel);
        bottomRegion.setStyle("-fx-background-color: lightblue; -fx-padding: 10px;");
        
        // Set the regions in the BorderPane
        borderPane.setTop(topRegion);
        borderPane.setLeft(leftRegion);
        borderPane.setCenter(centerRegion);
        borderPane.setRight(rightRegion);
        borderPane.setBottom(bottomRegion);
        
        // Create a scene with the BorderPane
        Scene scene = new Scene(borderPane, 400, 300);
        
        // Set the scene on the stage
        primaryStage.setTitle("Layout Example");
        primaryStage.setScene(scene);
        primaryStage.show();
    }
    
    public static void main(String[] args) {
        launch(args);
    }
}

UI Controls

JavaFX provides a rich set of UI controls:

  • Button: A standard push button
  • Label: A non-editable text control
  • TextField: A text input control
  • PasswordField: A text input control that masks the input
  • TextArea: A multi-line text input control
  • CheckBox: A control with a checked or unchecked state
  • RadioButton: A control that can be selected within a ToggleGroup
  • ComboBox: A drop-down list control
  • ListView: A list of selectable items
  • TableView: A table of data with columns
  • DatePicker: A control for selecting dates
  • Slider: A control for selecting a value from a range
  • ProgressBar: A control that visualizes progress
import javafx.application.Application;
import javafx.geometry.Insets;
import javafx.scene.Scene;
import javafx.scene.control.*;
import javafx.scene.layout.GridPane;
import javafx.stage.Stage;

public class ControlsExample extends Application {
    
    @Override
    public void start(Stage primaryStage) {
        // Create a GridPane layout
        GridPane grid = new GridPane();
        grid.setPadding(new Insets(10));
        grid.setHgap(10);
        grid.setVgap(10);
        
        // Add controls to the grid
        
        // Label and TextField
        grid.add(new Label("Name:"), 0, 0);
        TextField nameField = new TextField();
        grid.add(nameField, 1, 0);
        
        // Label and PasswordField
        grid.add(new Label("Password:"), 0, 1);
        PasswordField passwordField = new PasswordField();
        grid.add(passwordField, 1, 1);
        
        // CheckBox
        CheckBox rememberMeCheckBox = new CheckBox("Remember me");
        grid.add(rememberMeCheckBox, 1, 2);
        
        // RadioButtons with ToggleGroup
        grid.add(new Label("Gender:"), 0, 3);
        ToggleGroup genderGroup = new ToggleGroup();
        RadioButton maleRadio = new RadioButton("Male");
        maleRadio.setToggleGroup(genderGroup);
        RadioButton femaleRadio = new RadioButton("Female");
        femaleRadio.setToggleGroup(genderGroup);
        RadioButton otherRadio = new RadioButton("Other");
        otherRadio.setToggleGroup(genderGroup);
        
        GridPane radioGrid = new GridPane();
        radioGrid.setHgap(10);
        radioGrid.add(maleRadio, 0, 0);
        radioGrid.add(femaleRadio, 1, 0);
        radioGrid.add(otherRadio, 2, 0);
        grid.add(radioGrid, 1, 3);
        
        // ComboBox
        grid.add(new Label("Country:"), 0, 4);
        ComboBox countryComboBox = new ComboBox<>();
        countryComboBox.getItems().addAll("USA", "Canada", "UK", "Australia", "Germany");
        grid.add(countryComboBox, 1, 4);
        
        // DatePicker
        grid.add(new Label("Birth Date:"), 0, 5);
        DatePicker datePicker = new DatePicker();
        grid.add(datePicker, 1, 5);
        
        // Slider
        grid.add(new Label("Age:"), 0, 6);
        Slider ageSlider = new Slider(0, 100, 25);
        ageSlider.setShowTickLabels(true);
        ageSlider.setShowTickMarks(true);
        ageSlider.setMajorTickUnit(25);
        ageSlider.setMinorTickCount(5);
        ageSlider.setBlockIncrement(5);
        grid.add(ageSlider, 1, 6);
        
        // Button
        Button submitButton = new Button("Submit");
        submitButton.setOnAction(event -> {
            System.out.println("Name: " + nameField.getText());
            System.out.println("Remember Me: " + rememberMeCheckBox.isSelected());
            
            RadioButton selectedRadio = (RadioButton) genderGroup.getSelectedToggle();
            if (selectedRadio != null) {
                System.out.println("Gender: " + selectedRadio.getText());
            }
            
            System.out.println("Country: " + countryComboBox.getValue());
            System.out.println("Birth Date: " + datePicker.getValue());
            System.out.println("Age: " + (int) ageSlider.getValue());
        });
        grid.add(submitButton, 1, 7);
        
        // Create a scene with the grid
        Scene scene = new Scene(grid, 400, 350);
        
        // Set the scene on the stage
        primaryStage.setTitle("Controls Example");
        primaryStage.setScene(scene);
        primaryStage.show();
    }
    
    public static void main(String[] args) {
        launch(args);
    }
}

Event Handling

JavaFX uses an event-driven programming model. Events are generated when the user interacts with the UI, and event handlers respond to these events.

import javafx.application.Application;
import javafx.event.ActionEvent;
import javafx.event.EventHandler;
import javafx.geometry.Insets;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.control.Label;
import javafx.scene.control.TextField;
import javafx.scene.input.MouseEvent;
import javafx.scene.layout.VBox;
import javafx.stage.Stage;

public class EventHandlingExample extends Application {
    
    @Override
    public void start(Stage primaryStage) {
        // Create UI components
        Label messageLabel = new Label("Enter your name:");
        TextField nameField = new TextField();
        Button submitButton = new Button("Submit");
        Label resultLabel = new Label();
        
        // Method 1: Using an anonymous inner class
        submitButton.setOnAction(new EventHandler() {
            @Override
            public void handle(ActionEvent event) {
                String name = nameField.getText();
                resultLabel.setText("Hello, " + name + "!");
            }
        });
        
        // Method 2: Using a lambda expression (more concise)
        // submitButton.setOnAction(event -> {
        //     String name = nameField.getText();
        //     resultLabel.setText("Hello, " + name + "!");
        // });
        
        // Method 3: Using a named method reference
        // submitButton.setOnAction(this::handleSubmitButtonAction);
        
        // Mouse event example
        submitButton.setOnMouseEntered(event -> {
            submitButton.setStyle("-fx-background-color: lightblue;");
        });
        
        submitButton.setOnMouseExited(event -> {
            submitButton.setStyle("");
        });
        
        // Create a layout and add components
        VBox root = new VBox(10);
        root.setPadding(new Insets(10));
        root.getChildren().addAll(messageLabel, nameField, submitButton, resultLabel);
        
        // Create a scene with the layout
        Scene scene = new Scene(root, 300, 200);
        
        // Set the scene on the stage
        primaryStage.setTitle("Event Handling Example");
        primaryStage.setScene(scene);
        primaryStage.show();
    }
    
    // Method for Method 3 event handling
    private void handleSubmitButtonAction(ActionEvent event) {
        // Implementation here
    }
    
    public static void main(String[] args) {
        launch(args);
    }
}

FXML and Scene Builder

FXML is an XML-based language that provides a convenient way to define user interfaces. Scene Builder is a visual layout tool for designing JavaFX application user interfaces.

FXML Example

<?xml version="1.0" encoding="UTF-8"?>

<?import javafx.geometry.Insets?>
<?import javafx.scene.control.Button?>
<?import javafx.scene.control.Label?>
<?import javafx.scene.control.TextField?>
<?import javafx.scene.layout.VBox?>

<VBox spacing="10" xmlns:fx="http://javafx.com/fxml" fx:controller="com.example.FXMLExampleController">
    <padding>
        <Insets top="10" right="10" bottom="10" left="10"/>
    </padding>
    
    <Label text="Enter your name:"/>
    <TextField fx:id="nameField"/>
    <Button text="Submit" onAction="#handleSubmitButtonAction"/>
    <Label fx:id="resultLabel"/>
</VBox>

Controller Class

package com.example;

import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.scene.control.Label;
import javafx.scene.control.TextField;

public class FXMLExampleController {
    
    @FXML
    private TextField nameField;
    
    @FXML
    private Label resultLabel;
    
    @FXML
    private void handleSubmitButtonAction(ActionEvent event) {
        String name = nameField.getText();
        resultLabel.setText("Hello, " + name + "!");
    }
}

import javafx.application.Application;

Canvas Example

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.canvas.Canvas;
import javafx.scene.canvas.GraphicsContext;
import javafx.scene.layout.StackPane;
import javafx.scene.paint.Color;
import javafx.stage.Stage;

public class CanvasExample extends Application {
    
    @Override
    public void start(Stage primaryStage) {
        // Create a canvas
        Canvas canvas = new Canvas(400, 400);
        
        // Get the graphics context
        GraphicsContext gc = canvas.getGraphicsContext2D();
        
        // Draw shapes
        drawShapes(gc);
        
        // Create a layout and add the canvas
        StackPane root = new StackPane();
        root.getChildren().add(canvas);
        
        // Create a scene with the layout
        Scene scene = new Scene(root, 400, 400);
        
        // Set the scene on the stage
        primaryStage.setTitle("Canvas Example");
        primaryStage.setScene(scene);
        primaryStage.show();
    }
    
    private void drawShapes(GraphicsContext gc) {
        // Set line width
        gc.setLineWidth(2.0);
        
        // Draw a rectangle
        gc.setStroke(Color.RED);
        gc.strokeRect(50, 50, 100, 100);
        
        // Fill a rectangle
        gc.setFill(Color.BLUE);
        gc.fillRect(200, 50, 100, 100);
        
        // Draw a rounded rectangle
        gc.setStroke(Color.GREEN);
        gc.strokeRoundRect(50, 200, 100, 100, 20, 20);
        
        // Fill a rounded rectangle
        gc.setFill(Color.ORANGE);
        gc.fillRoundRect(200, 200, 100, 100, 20, 20);
        
        // Draw a circle
        gc.setStroke(Color.PURPLE);
        gc.strokeOval(100, 100, 50, 50);
        
        // Fill an oval
        gc.setFill(Color.CYAN);
        gc.fillOval(250, 250, 80, 40);
        
        // Draw a line
        gc.setStroke(Color.BLACK);
        gc.strokeLine(50, 50, 350, 350);
        
        // Draw text
        gc.setFill(Color.BLACK);
        gc.fillText("Hello, Canvas!", 150, 30);
    }
    
    public static void main(String[] args) {
        launch(args);
    }
}
