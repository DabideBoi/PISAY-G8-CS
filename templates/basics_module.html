Introduction to C++

C++ is a powerful general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language. It offers a combination of high-level and low-level language features, making it suitable for system programming, embedded systems, game development, and more.

Basic Syntax

Every C++ program has at least one function, the main() function, which is the entry point of the program:

#include <iostream>
using namespace std;

int main() {
    // Your code here
    return 0;
}

Comments

Comments are used to explain code and are ignored by the compiler:

// This is a single-line comment

/* This is a
   multi-line comment */

Input and Output

C++ uses the iostream library for input and output operations:

#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!"; // Output
    
    int num;
    cin >> num; // Input
    
    return 0;
}

Variables and Data Types

Variables are containers for storing data values. In C++, variables must be declared with a specific type:

int myNum = 5;               // Integer
double myFloatingNum = 5.99;  // Floating point number
char myLetter = 'D';          // Character
string myText = "Hello";      // String
bool myBoolean = true;        // Boolean

Constants

Constants are variables whose values cannot be changed after declaration:

const int MINUTES_PER_HOUR = 60;
const float PI = 3.14159;

Operators

C++ provides various operators for performing operations on variables and values:

Arithmetic Operators

int a = 10, b = 5;
int sum = a + b;      // Addition: 15
int diff = a - b;     // Subtraction: 5
int product = a * b;  // Multiplication: 50
int quotient = a / b; // Division: 2
int remainder = a % b; // Modulus: 0

Assignment Operators

int x = 10;
x += 5;  // x = x + 5
x -= 3;  // x = x - 3
x *= 2;  // x = x * 2
x /= 4;  // x = x / 4
x %= 3;  // x = x % 3

Comparison Operators

int a = 5, b = 10;
bool equal = (a == b);       // Equal to: false
bool notEqual = (a != b);    // Not equal to: true
bool greater = (a > b);      // Greater than: false
bool less = (a < b);         // Less than: true
bool greaterEqual = (a >= b); // Greater than or equal to: false
bool lessEqual = (a <= b);    // Less than or equal to: true

Logical Operators

bool a = true, b = false;
bool andResult = a && b;  // Logical AND: false
bool orResult = a || b;   // Logical OR: true
bool notResult = !a;      // Logical NOT: false

Type Casting

Type casting is when you convert a variable from one data type to another:

// Implicit casting
int myInt = 9;
double myDouble = myInt;  // Automatic casting: int to double

// Explicit casting
double myDouble = 9.78;
int myInt = (int) myDouble;  // Manual casting: double to int
