Introduction to Python Data Structures

Python provides several built-in data structures that allow you to store, organize, and manipulate data efficiently. Understanding these data structures is essential for writing effective Python code.

Lists

Lists are ordered, mutable collections that can store elements of different types:

# Creating a list
my_list = [1, 2, 3, 'Python', True]

# Accessing elements (indexing starts at 0)
print(my_list[0])  # Output: 1
print(my_list[-1])  # Output: True (negative indexing starts from the end)

# Modifying elements
my_list[0] = 100
print(my_list)  # Output: [100, 2, 3, 'Python', True]

# Common list methods
my_list.append(6)  # Add an element to the end
my_list.insert(1, 'inserted')  # Insert at specific position
my_list.remove('Python')  # Remove by value
popped_value = my_list.pop()  # Remove and return the last element
my_list.sort()  # Sort the list (elements must be of comparable types)
my_list.reverse()  # Reverse the list

Tuples

Tuples are ordered, immutable collections that can store elements of different types:

# Creating a tuple
my_tuple = (1, 2, 3, 'Python', True)

# Accessing elements
print(my_tuple[0])  # Output: 1
print(my_tuple[-1])  # Output: True

# Attempting to modify a tuple will result in an error
# my_tuple[0] = 100  # TypeError: 'tuple' object does not support item assignment

# Tuple methods
print(my_tuple.count(1))  # Count occurrences of a value
print(my_tuple.index('Python'))  # Find the index of a value

Dictionaries

Dictionaries are mutable, unordered collections of key-value pairs:

# Creating a dictionary
my_dict = {'name': 'John', 'age': 30, 'city': 'New York'}

# Accessing values
print(my_dict['name'])  # Output: John
print(my_dict.get('age'))  # Output: 30 (safer method)

# Modifying values
my_dict['age'] = 31
my_dict['email'] = 'john@example.com'  # Add new key-value pair
print(my_dict)  # Output: {'name': 'John', 'age': 31, 'city': 'New York', 'email': 'john@example.com'}

# Dictionary methods
print(my_dict.keys())  # Get all keys
print(my_dict.values())  # Get all values
print(my_dict.items())  # Get all key-value pairs
my_dict.pop('city')  # Remove a key-value pair

Sets

Sets are unordered, mutable collections of unique elements:

# Creating a set
my_set = {1, 2, 3, 3, 4}  # Duplicate values are automatically removed
print(my_set)  # Output: {1, 2, 3, 4}

# Set operations
set1 = {1, 2, 3}
set2 = {3, 4, 5}

# Union: elements in either set
print(set1.union(set2))  # Output: {1, 2, 3, 4, 5}
# or using the | operator
print(set1 | set2)  # Output: {1, 2, 3, 4, 5}

# Intersection: elements in both sets
print(set1.intersection(set2))  # Output: {3}
# or using the & operator
print(set1 & set2)  # Output: {3}

# Difference: elements in set1 but not in set2
print(set1.difference(set2))  # Output: {1, 2}
# or using the - operator
print(set1 - set2)  # Output: {1, 2}

# Symmetric difference: elements in either set, but not in both
print(set1.symmetric_difference(set2))  # Output: {1, 2, 4, 5}
# or using the ^ operator
print(set1 ^ set2)  # Output: {1, 2, 4, 5}

List Comprehensions

List comprehensions provide a concise way to create lists:

# Creating a list of squares
squares = [x**2 for x in range(10)]
print(squares)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# List comprehension with condition
even_squares = [x**2 for x in range(10) if x % 2 == 0]
print(even_squares)  # Output: [0, 4, 16, 36, 64]

# Dictionary comprehension
dict_comp = {x: x**2 for x in range(5)}
print(dict_comp)  # Output: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# Set comprehension
set_comp = {x**2 for x in range(5)}
print(set_comp)  # Output: {0, 1, 4, 9, 16}

When to Use Each Data Structure

  • Lists: Use when you need an ordered collection that might change.
  • Tuples: Use when you need an ordered collection that should not change.
  • Dictionaries: Use when you need to associate values with keys for fast lookup.
  • Sets: Use when you need to store unique values and perform set operations.
