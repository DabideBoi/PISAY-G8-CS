Introduction to Python

Python is a high-level, interpreted programming language known for its readability and simplicity. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability with its notable use of significant whitespace.

Basic Syntax

Python uses indentation to define code blocks, unlike many other languages that use braces:

# A simple Python program
def greet(name):
    # This is a function that greets the person passed in as a parameter
    print(f"Hello, {name}!")
    
# Function call
greet("World")

Comments

Comments in Python start with the hash character (#) and extend to the end of the line:

# This is a single-line comment

"""
This is a multi-line comment or docstring.
It can span multiple lines.
"""

Input and Output

Python uses the print() function for output and input() function for input:

# Output
print("Hello, World!")

# Input
name = input("Enter your name: ")
print(f"Hello, {name}!")

Variables and Data Types

Python is dynamically typed, meaning you don't need to declare the type of a variable:

x = 5               # Integer
y = 5.99            # Float
name = "John"       # String
is_valid = True     # Boolean

Common Data Types

  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • None Type: NoneType

Operators

Python provides various operators for performing operations on variables and values:

Arithmetic Operators

a = 10
b = 5
print(a + b)    # Addition: 15
print(a - b)    # Subtraction: 5
print(a * b)    # Multiplication: 50
print(a / b)    # Division: 2.0
print(a % b)    # Modulus: 0
print(a ** b)   # Exponentiation: 100000
print(a // b)   # Floor division: 2

Comparison Operators

a = 10
b = 5
print(a == b)   # Equal: False
print(a != b)   # Not equal: True
print(a > b)    # Greater than: True
print(a < b)    # Less than: False
print(a >= b)   # Greater than or equal to: True
print(a <= b)   # Less than or equal to: False

Logical Operators

a = True
b = False
print(a and b)  # Logical AND: False
print(a or b)   # Logical OR: True
print(not a)    # Logical NOT: False

Control Flow

Python uses indentation to define blocks of code in control flow statements:

If-Else Statement

x = 10

if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is equal to 5")
else:
    print("x is less than 5")

Loops

# For loop
for i in range(5):
    print(i)  # Prints 0, 1, 2, 3, 4

# While loop
count = 0
while count < 5:
    print(count)
    count += 1
