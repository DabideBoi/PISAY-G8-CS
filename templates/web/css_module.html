Introduction to CSS

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS Syntax

A CSS rule consists of a selector and a declaration block:

selector {
    property: value;
    property: value;
}

Example

h1 {
    color: blue;
    font-size: 24px;
}

Adding CSS to HTML

There are three ways to insert CSS into your HTML:

1. Inline CSS

Using the style attribute within HTML elements:

<h1 style="color: blue; font-size: 24px;">This is a heading</h1>

2. Internal CSS

Using the <style> element in the <head> section:

<head>
    <style>
        h1 {
            color: blue;
            font-size: 24px;
        }
    </style>
</head>

3. External CSS

Using an external CSS file:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

With the following content in styles.css:

h1 {
    color: blue;
    font-size: 24px;
}

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Element Selector

Selects all elements with the specified element name:

p {
    color: red;
}

ID Selector

Selects an element with a specific id attribute:

#header {
    background-color: lightblue;
}

Class Selector

Selects all elements with a specific class attribute:

.highlight {
    background-color: yellow;
}

Descendant Selector

Selects all elements that are descendants of a specified element:

div p {
    color: green;
}

Child Selector

Selects all elements that are direct children of a specified element:

div > p {
    color: blue;
}

Adjacent Sibling Selector

Selects an element that is directly after another specific element:

h1 + p {
    margin-top: 0;
}

Attribute Selector

Selects elements with a specific attribute:

input[type="text"] {
    border: 1px solid gray;
}

Pseudo-classes

Selects elements based on a certain state:

a:hover {
    color: red;
}

li:first-child {
    font-weight: bold;
}

CSS Box Model

All HTML elements can be considered as boxes. The CSS box model describes the rectangular boxes that are generated for elements in the document tree and laid out according to the visual formatting model.

The box model consists of:

  • Content - The content of the box, where text and images appear
  • Padding - Clears an area around the content (inside the box)
  • Border - A border that goes around the padding and content
  • Margin - Clears an area outside the border
div {
    width: 300px;
    padding: 10px;
    border: 5px solid gray;
    margin: 20px;
}

Total width = width + left padding + right padding + left border + right border + left margin + right margin

CSS Colors

Colors in CSS can be specified in several ways:

Color Names

h1 {
    color: red;
}

RGB Values

h1 {
    color: rgb(255, 0, 0);  /* Red */
}

RGBA Values (RGB with Alpha/Transparency)

h1 {
    color: rgba(255, 0, 0, 0.5);  /* Semi-transparent red */
}

HEX Values

h1 {
    color: #ff0000;  /* Red */
}

HSL Values (Hue, Saturation, Lightness)

h1 {
    color: hsl(0, 100%, 50%);  /* Red */
}

CSS Typography

CSS provides several properties for styling text:

p {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-weight: bold;
    font-style: italic;
    text-align: center;
    text-decoration: underline;
    line-height: 1.5;
    letter-spacing: 1px;
    text-transform: uppercase;
}

CSS Layout

CSS provides several techniques for laying out elements on a page:

Display Property

div {
    display: block;  /* Default for div */
}

span {
    display: inline;  /* Default for span */
}

nav li {
    display: inline-block;  /* Combines features of both */
}

.hidden {
    display: none;  /* Removes the element */
}

Position Property

div {
    position: static;  /* Default */
}

.relative {
    position: relative;
    top: 10px;
    left: 20px;
}

.absolute {
    position: absolute;
    top: 50px;
    left: 50px;
}

.fixed {
    position: fixed;
    bottom: 20px;
    right: 20px;
}

.sticky {
    position: sticky;
    top: 0;
}

Float Property

.left {
    float: left;
}

.right {
    float: right;
}

.clear {
    clear: both;
}

Flexbox

.container {
    display: flex;
    flex-direction: row;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
}

.item {
    flex: 1;
}

Grid

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
}

.item-a {
    grid-column: 1 / 3;
    grid-row: 1;
}

.item-b {
    grid-column: 3;
    grid-row: 1 / 3;
}

CSS Responsive Design

Responsive web design makes your web page look good on all devices:

Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

/* For desktop */
.container {
    width: 960px;
}

/* For tablet */
@media (max-width: 768px) {
    .container {
        width: 100%;
    }
}

/* For mobile */
@media (max-width: 480px) {
    .container {
        padding: 10px;
    }
}

CSS Transitions and Animations

Transitions

button {
    background-color: blue;
    transition: background-color 0.3s ease;
}

button:hover {
    background-color: darkblue;
}

Animations

@keyframes slide {
    0% {
        transform: translateX(0);
    }
    100% {
        transform: translateX(100px);
    }
}

.box {
    animation: slide 2s ease infinite alternate;
}
