Introduction to JavaScript

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. JavaScript is one of the core technologies of the World Wide Web, alongside HTML and CSS. It enables interactive web pages and is an essential part of web applications.

Basic Syntax

JavaScript statements are commands to the browser and are separated by semicolons:

// A simple JavaScript program
function greet(name) {
    // This is a function that greets the person passed in as a parameter
    console.log(`Hello, ${name}!`);
}

// Function call
greet("World");

Comments

JavaScript supports both single-line and multi-line comments:

// This is a single-line comment

/* 
   This is a multi-line comment
   It can span multiple lines
*/

Output

JavaScript can "display" data in different ways:

// Writing to the browser console
console.log("Hello, World!");

// Writing to an HTML element
document.getElementById("demo").innerHTML = "Hello, World!";

// Writing to an alert box
alert("Hello, World!");

Variables and Data Types

JavaScript has three ways to declare variables:

// Using var (function-scoped, older way)
var x = 5;

// Using let (block-scoped, preferred for variables that can change)
let y = 5.99;

// Using const (block-scoped, for constants that cannot be reassigned)
const PI = 3.14159;

Common Data Types

  • Primitive Types: String, Number, Boolean, undefined, null, Symbol, BigInt
  • Reference Types: Object, Array, Function, Date, RegExp
// String
let name = "John";

// Number
let age = 30;
let price = 19.99;

// Boolean
let isActive = true;

// Array
let colors = ["red", "green", "blue"];

// Object
let person = {
    firstName: "John",
    lastName: "Doe",
    age: 30
};

Operators

JavaScript provides various operators for performing operations on variables and values:

Arithmetic Operators

let a = 10;
let b = 5;
console.log(a + b);    // Addition: 15
console.log(a - b);    // Subtraction: 5
console.log(a * b);    // Multiplication: 50
console.log(a / b);    // Division: 2
console.log(a % b);    // Modulus: 0
console.log(a ** b);   // Exponentiation: 100000

Comparison Operators

let a = 10;
let b = 5;
console.log(a == b);   // Equal to: false
console.log(a === b);  // Equal value and type: false
console.log(a != b);   // Not equal: true
console.log(a !== b);  // Not equal value or type: true
console.log(a > b);    // Greater than: true
console.log(a < b);    // Less than: false
console.log(a >= b);   // Greater than or equal to: true
console.log(a <= b);   // Less than or equal to: false

Logical Operators

let a = true;
let b = false;
console.log(a && b);  // Logical AND: false
console.log(a || b);  // Logical OR: true
console.log(!a);      // Logical NOT: false

Control Flow

JavaScript provides various control flow statements:

If-Else Statement

let x = 10;

if (x > 5) {
    console.log("x is greater than 5");
} else if (x === 5) {
    console.log("x is equal to 5");
} else {
    console.log("x is less than 5");
}

Loops

// For loop
for (let i = 0; i < 5; i++) {
    console.log(i);  // Prints 0, 1, 2, 3, 4
}

// While loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// For...of loop (for iterables like arrays)
const colors = ["red", "green", "blue"];
for (const color of colors) {
    console.log(color);
}

// For...in loop (for object properties)
const person = { name: "John", age: 30 };
for (const key in person) {
    console.log(`${key}: ${person[key]}`);
}
